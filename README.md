# Brawl Stars FC

Proyecto de aprendizaje y prácticas de desarrollo web. No está afiliado ni aprobado por Supercell.

## Stack

- **Framework:** Astro
- **CSS:** Tailwind CSS
- **Package manager:** pnpm

## Propósito

Este proyecto se usa únicamente con fines educativos para practicar desarrollo web, diseño de interfaces y experimentación técnica.
